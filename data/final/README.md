# Final Data Inputs

The default replication run starts from prebuilt final datasets:

- `final_dataset.dta`
- `final_dataset.fst`
- `final_dataset_unfiltered.fst`

This package currently links those files to the existing project build so the default run can begin from the already-prepared input artifact.

Rebuilding these files from raw data is optional and is not part of the default run.
